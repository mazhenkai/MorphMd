# Template 目录说明

存放文档模板，目前包含封面模板。

## 目录结构

```
Template/
└── Cover/                  # 封面模板
    ├── Samples/            # 示例（无敏感信息）
    │   ├── 示例封面.md     # 封面模板文件（含占位符）
    │   └── 示例封面.json   # 封面数据文件
    └── SelfRunningFiles/   # 实际项目封面（已 gitignore）
```

## 封面使用方法

### 1. 准备文件

每个封面由两个同名文件组成：

- `封面名.md` — 模板，使用 `{{字段名}}` 占位符
- `封面名.json` — 数据，填写实际字段值

### 2. 支持的字段

| 字段 | 说明 |
|------|------|
| `{{项目名称}}` | 主标题 |
| `{{文档标题}}` | 副标题 |
| `{{副标题}}` | 顶部小字 |
| `{{版本}}` | 文档版本 |
| `{{日期}}` | 文档日期 |
| `{{环境}}` | 部署环境 |
| `{{状态}}` | 发布状态 |
| `{{联系人}}` | 负责人姓名 |
| `{{联系方式}}` | 联系邮箱 |
| `{{最后更新}}` | 最后更新时间 |
| `{{备注}}` | 底部备注 |

### 3. 运行命令

```bash
python Bin/add_cover.py <封面md路径> <内容pdf路径>
```

示例：

```bash
python Bin/add_cover.py \
  Template/Cover/Samples/示例封面.md \
  Output/pdf/Tests/文档.pdf
```

输出文件自动命名为 `文档_covered.pdf`，保存在同目录下，不覆盖原文件。
